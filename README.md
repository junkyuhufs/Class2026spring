---

Welcome! This repository supports the course **NLP for English Teachers** in the Graduate School of Education. 👋

English teachers constantly work with **texts, vocabulary, grammar patterns, learner writing, feedback, and classroom materials**.
For that reason, coding and NLP are not just technical skills—they can become practical tools for teaching. 📚

With basic Python and NLP, teachers can begin to:

* explore vocabulary and lexical chunks more systematically 🔎
* identify useful language patterns in texts
* create and adapt classroom materials more efficiently ✏️
* support reading, writing, grammar, speaking, and listening activities 🎯
* reduce repetitive work and focus more on pedagogy
* turn teaching ideas into small digital tools 💡

In other words, **coding and NLP can help English teachers become not only users of materials, but also designers of better learning experiences**. 🌱

This course is designed for English teachers who want to learn how **basic Python, NLP, and GenAI** can support classroom practice.
The goal is not to become a programmer, but to learn how to work with texts and teaching materials in more systematic, creative, and pedagogically meaningful ways.

## What we will do in this course 🚀

In this course, we will:

* learn basic Python 🐍
* explore classroom-oriented NLP
* use **GitHub** to organize materials and code
* run Python in **Google Colab** 💻
* build toward a simple **teaching-support app**
* use **GenAI** as a partner for planning, debugging, and revision 🤝

Our main idea is simple:

> **Theory to Tool**
> turning ideas from English education and SLA into usable classroom tools

## First class 🧭

The first class is mainly an orientation.
We will look at the overall direction of the course, preview possible tools and projects, and see how GitHub, Colab, and GenAI will be used throughout the semester.
